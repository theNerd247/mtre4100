\begin{abstract}
	This lab investigates the basic properties of creating a Fourier transform of
	a square wave. We show how to create the coefficients using integration
	techniques and the effects of filtering on input signals. Finally we show how
	the fourier transform breaks down a complex periodic signal to a set of simple
	harmonic functions.
\end{abstract}
